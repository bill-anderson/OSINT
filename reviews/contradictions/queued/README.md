# reviews/contradictions/queued/ — staging area

This folder holds **provenance-hunt briefs migrated out of `reviews/gaps.md` on 2026-07-20**.
They are contradiction-shaped items rather than structural gaps: each names a claim the wiki
holds or asserts that needs a **dated primary**, a **disputed value settled**, or a **status
confirmed** — work a reconcile pass can actually do.

They are promoted into `open/` in **country or batch tranches**, so a single reconcile pass is
not swamped by several dozen items at once. **Nothing here is worked until it is promoted** —
`open/` remains the only live worklist.
