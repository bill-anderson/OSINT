# Contradiction: where is Somalia's national-ID (SNID) data actually hosted — inside Somalia or in Pakistan?

**Opened:** 2026-07-16 (SLE·SOM·SSD·STP batch)

## The dispute

Somalia's foundational national digital ID (SNID) is being built with technology from Pakistan's NADRA. Sources disagree on **where the biometric/identity data is physically hosted and who controls it** — a core data-sovereignty question.

- **Claim A — data is stored securely within Somalia, under Somali control.**
  - Asserted by: NIRA Director-General Abdiwali Ali Abdulle (a.k.a. Abdiwali Timacadde), via Hiiraan Online (2025-11-17).
  - Source URL held: https://www.hiiraan.com/news4/2025/Nov/203644/a_smart_national_id_powers_somalia_s_digital_public_infrastructure_dpi_vision.aspx
- **Claim B — the system is based in Pakistan and managed by foreign workers.**
  - Asserted by: Garowe Online reporting on personal-data-security concerns in the ID rollout (2025, page undated — date is a proxy).
  - Source URL held: https://garoweonline.com/en/news/somalia/somalia-growing-concerns-over-personal-data-security-in-national-id-rollout

## Likely reconciliation to test

Establish, with an as-of date: (1) where SNID enrolment/biometric data is physically stored (on-shore Somali data centre vs NADRA/Pakistan-hosted); (2) who administers the production environment (Somali NIRA staff vs NADRA personnel); (3) whether a data-localisation or hosting clause exists in the NADRA arrangement. The two claims may both be partly true at different stages (e.g. NADRA-built and initially NADRA-operated, migrating on-shore) — pin the current state and date it.

## Instruction

These sources give different accounts of where Somalia's national-ID data is hosted and who controls it. Investigate the discrepancy and suggest a resolution, recording an as-of date for each claim.
