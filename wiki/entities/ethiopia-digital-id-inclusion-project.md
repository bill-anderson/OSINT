# Ethiopia Digital ID for Inclusion and Services Project (World Bank, P179040)

World Bank project (**P179040**) providing **$350 million** to Ethiopia's **National ID Program (NIDP)** to roll out the **[[fayda-ethiopia-national-id|Fayda]]** digital ID to **90 million people, including refugees and IDPs**. Launched 2024, it targets inclusion for those currently lacking identification (a barrier to the formal economy), reduced double benefit-collection and ghost accounts, streamlined safety-net payments, and greater trust in digital services. Over 12m registered by Feb 2025. A core financier behind Ethiopia's flagship **[[mosip]]**-based foundational-ID rollout.

- **Refugee & displaced-person inclusion (2026).** A joint task force supported ~**190,000 refugee
  registrations**, **170,000 refugee ID issuances** and **200,000+ host-community registrations**; UNHCR
  endorsed Fayda for refugee service access (2026-01). The new **"Makatet"** (Amharic: *inclusion*) framework
  (launched mid-2026) moves Ethiopia from an aid-dependent encampment model toward **state-led development and
  self-reliance** for displaced people — **~1.1m refugees from 27 countries + 2m+ IDPs** — with Fayda issuance a
  key priority. Off-grid reach uses **Integrated Biometrics "Kojak"** scanners for rural enrollment. Sources:
