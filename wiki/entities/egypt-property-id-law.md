# Egypt Property ID Law

Law approved by Egypt's House of Representatives (**27 April 2025**; introduced 2023) mandating a **unique digital ID for every property** and a centralised database linking real-estate assets to a government platform. Each property ID records **location, use, ownership, licensing, violations and transaction history**, tied to a **unified digital base map** and secure spatial information system, with physical ID plates on properties.

**Property IDs become a prerequisite** for real-estate-registry/notary registration and for utility connections (water/electricity/gas). Tampering: EGP 5,000–30,000 fines or up to six months' prison; national-security properties exempt. The **[[dpi.registry|land-registry]]** leg of Egypt's DPI.
