# Innovendi Data Technologies

Private **port-data / connectivity company** behind the **Douala Smart Port** build — notable for a
**vendor-financed** model that inverts the usual sovereignty-relevant pattern.

## Activity in this wiki

- **Douala Port 5G / edge-cloud (Nov 2025).** Sealed a private **4G/5G + IoT + edge-cloud** agreement at
  Cameroon's busiest port in which **INNOVENDI pays the Port Authority of Douala (PAD)** — vendor pays
  operator, not the reverse. Source: [[2025-11-28-douala-port-5g-innovendi-agreement-itweb]].
- **CNPS takes equity (Mar 2026).** The state pension fund **CNPS entered the capital of Innovendi Data
  Technologies** — a new financing pattern for CMR digital infrastructure (public pension money into a
  private port-data firm). Source: [[2026-03-04-leconomie-cnps-innovendi-smart-port]].
