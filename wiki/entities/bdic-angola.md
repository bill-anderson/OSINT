# Base de Dados de Identificação Civil (BDIC) — Angola

Angola's **civil-identification database** — the foundational register underpinning the
**[[bilhete-de-identidade-angola|BI]]**. It is being made interoperable with the
adult-citizen database (**[[base-de-dados-de-cidadaos-maiores-angola|BDCM]]**) so that the
BDCM can be consolidated as the official electoral reference base, biometrically
de-duplicated and residence-updated.

## Current state

- The 2026-06-08 presidential decree names the **stabilisation and permanence of data
  interoperability between the BDIC and the BDCM** as an explicit task of the proof-of-life
  commission — the BDIC (civil identity) feeding and reconciling with the BDCM (electoral
  reference) is the technical core of Angola's registry-integration push. Source:
  [[2026-06-08-pr-cria-comissao-actualizacao-dados-cidadaos-maiores-bdcm]].

Fuller treatment of the interoperability thread is on **[[angola--dpi-registry]]**.
