# SITA Passenger Data System — Angola (API/PNR Gateway, Luanda)

A **border-data-exchange deployment** by aviation-IT vendor **SITA** (Société
Internationale de Télécommunications Aéronautiques) at Angola's **Dr. António Agostinho
Neto International Airport** (Luanda) — a working state data-exchange capability built for
border surveillance, notably ahead of the civil-ID system.

> **Note — acronym collision:** the source frontmatter tags `[[sita]]`, but the existing
> `sita` entity page is **South Africa's State Information Technology Agency**, a different
> organisation. The vendor here is the aviation-IT firm. Flagged for the orchestrator; no
> `[[sita]]` link is used below to avoid misdirection.

## What it is (2025-12)

- SITA's **Advance Passenger Information + Passenger Name Record (API PNR) Gateway** lets
  airlines transmit passport, identity, itinerary and booking data to Angolan authorities
  **before departure**; unauthorised passengers can be blocked from boarding Angola-bound
  flights.
- A new **Passenger Information Monitoring and Management Center** in Luanda analyses the
  data in real time to flag irregular travel patterns, with a **dedicated data centre**
  next door for secure processing — supported by SITA's Passenger Information Unit (PIU)
  services.
- Framed as bringing Angola in line with **UN Security Council, ICAO and EU** requirements,
  within a broader border-modernisation programme positioning the country as a regional
  transport hub (airport capacity 15m passengers/yr).
