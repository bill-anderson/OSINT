# Core-entities watchlist

Human-owned. The entities and instruments that always warrant an **eagerly
maintained** page — minted on first mention rather than waiting for the
≥3-source materiality bar. See CLAUDE.md → *Entities: tag always, page when
material*.

**How it's used.** During ingest an entity here gets a page created on first
mention; everything off this list waits until it is cited by ≥3 sources (or is a
domain-central reference instrument). The periodic **entity pass** consults this
list when minting and refreshing pages. Nothing else about tagging changes —
every named entity is still tagged in source frontmatter regardless of this list.

**Curating it.** Add an entity when it becomes something you want tracked from its
first appearance; remove one when it no longer merits eager upkeep. Keep it to the
few dozen that genuinely carry analytical weight — a long watchlist just recreates
the cost we're trying to avoid. Match each entry to the canonical entity slug where
a page already exists.
