# Paratus Group

Pan-African, African-owned network operator (16 countries across sub-equatorial
Africa; PoPs in Europe, UK, US), building regional terrestrial and data-centre
infrastructure.

## Activity in this wiki

- **Botswana–Zimbabwe–Zambia fibre corridor (2026-07).** With **Powertel**
  (ZESA's telecom arm), Paratus Zimbabwe switched on **phase 1 (Plumtree–Bulawayo)**
  of a cross-border digital spine; phase 2 (Bulawayo–Livingstone, Zambia) lights
  up **Sept 2026**. Deal: [[powertel-paratus-fibre-corridor]]. See [[ZWE]],
  [[infra.connect]], [[infra.capacity]]. Source:
  [[2026-07-06 Powertel, Paratus Zimbabwe switch on new digital highway]].
- **Namibia's first private mobile network (2025-09).** Paratus launched **Namibia's first privately-owned
  mobile network** (2 Sep 2025; LTE + 5G, data-only) — a challenger to the state duopoly [[mtc-namibia|MTC]]
  (>90% share) and [[telecom-namibia|Telecom Namibia]]. Namibia is "the group's power base": **N$1.42bn invested
  since 2018** plus **N$600m** on the mobile build (with **Nokia** + **Cerillion**). See [[NAM]],
  [[namibia--infra-connect]], [[infra.connect]]. Sources:
  [[2025-09-04-paratus-launches-namibia-first-private-mobile-network-connectingafrica]],
  [[2025-08-19-paratus-to-launch-mobile-network-namibiansun]].
- **Namibia data centres.** Runs the **Armada** commercial data centre in Namibia
  — context to Namibia's sovereign-DC ambitions. See [[NAM]]. Source:
  [[2026-06-15 Namibia encouraged by Angola on datacentre establishment]].
- **Angola: three-DC Luanda campus (ex-ITA).** DC1 (2017, ~1,500 servers) + DC2
  (2019, ~7,000 servers), plus a teleport and a OneWeb ground station; **DC3** (announced
  2023) is set to be **Angola's first Tier IV** facility (30,000 m², >10 MW / 2,000 racks) —
  financed by a **$31m EAIF (PIDG)/Ninety One** facility that also funds three regional fibre
  routes. See [[AGO]], [[angola--infra-store]]. Source:
  [[2023-06-27 Paratus to build third Angola data center (DC3, Tier IV)]].

See [[ZWE]], [[NAM]], [[AGO]], [[infra.connect]], [[angola--infra-store]].
