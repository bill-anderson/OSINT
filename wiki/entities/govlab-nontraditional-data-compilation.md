# The Re-Use of Non-Traditional Data for Public Interest Purposes

A curated compilation by **[[the-govlab]]** (Adam Zable and **[[stefaan-verhulst]]**, June 2026) of
**100 use cases (2024–2026)** in which non-traditional data — mobile-network and geolocation data,
social-media/interaction data, wearable and biomedical data, consumer/financial-transaction data,
open-web and human–AI-interaction data — has been re-used for public-interest purposes. A standing
reference for the "data collaboratives" model; filed with `status: needs-review` per the wiki's
bibliography rule — individual use cases are not promoted at intake. Reference document:
`2026-07-11 The Re-Use of  Non-Traditional Data  for Public Interest Purposes.pdf`.

## Why it is here

- A recurring reference on **alternative data sources for development/governance** — directly
  relevant to Africa's official-statistics funding crisis (cf. [[2026-07-11 data-systems-crossroads]],
  [[paris21]]) and to debates over mobile-money, telecom and satellite data as public-interest data
  assets ([[data.open]], [[data.statistics]]).
- Categorises use cases by data type and public-interest domain (health, mobility, financial
  inclusion, crisis response), with governance/consent caveats.
