# TECH5

Biometric and digital-identity technology provider (face, iris, fingerprint). Provides the **biometric onboarding/security layer** for Djibouti's **[[smart-wallet-djibouti|Smart Wallet]]**; **[[visa]]** formalised a **seven-year collaboration** with TECH5 in September 2025. Participated in the 15–16 September 2025 Visa technical mission to Djibouti.

- **Guinea national biometric ID (2026-03).** [[GIN]] TECH5 (with [[mosip]]) underpins Guinea's biometric national-ID and birth-certificate stack ([[oneci-guinea]]), rolled out from communes such as Sanoyah. See [[dpi.id]]. Source: [[2026-03-04 Guinea Launches Biometric Birth Certificates in Sanoyah in Civil Registry Modernization Push]].
- **Mauritania Houwiyeti digital ID (2022 pilot → 2024).** [[MRT]] TECH5 won Mauritania's UNDP-backed digital-ID pilot (June 2022, with local integrator SmartMS), providing selfie-based verification (T5-OmniMatch Face, T5-AirSnap, liveness) — the technology base for the **[[houwiyeti]]** app. See [[dpi.id]], [[dpi.govtech]]. Source: [[2024-08-15 Mauritania Launches Digital ID for e-Services, Built on TECH5 Platform (Houwiyeti)]].
