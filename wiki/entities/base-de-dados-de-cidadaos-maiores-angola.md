# Base de Dados de Cidadãos Maiores (BDCM) — Angola

Angola's **adult-citizen database** — a system for continuous registration and updating of
data on citizens aged 18+, serving as the **official proof-of-life instrument** for
electoral and administrative purposes. It is being consolidated as the **official
electoral reference base** handed to the electoral commission, via interoperability with
the civil-identification database (**[[bdic-angola|BDIC]]**). Run by the
**[[ministerio-da-administracao-do-territorio-angola|MAT]]**.

## Current state

- **Presidential commission to update it (2026-06-08).** A presidential decree created an
  interministerial commission (chaired by the Casa Civil chief **[[dionisio-da-fonseca]]**,
  with Interior, Foreign Affairs, Finance, MAT, Justice and MINTTICS) to run a biometric,
  in-person **proof-of-life** update — mandatory in pre-electoral periods since 2025 — to
  purge deceased or rights-suspended names. Named tasks: biometric de-duplication, updated
  residence, diaspora validation via consulates, hardening BUAP energy/computing/comms/
  digital security, and **stabilising interoperability between the BDIC and the BDCM**.
  Source: [[2026-06-08-pr-cria-comissao-actualizacao-dados-cidadaos-maiores-bdcm]].
- **Funded by a $275m supplementary credit (2026-02)** to MAT for the BDCM proof-of-life
  action plan. Source: [[2026-02-26-credito-suplementar-275-milhoes-usd-mat-bdcm]].
- **Target universe: 16,707,455 citizens**, to be captured across the BUAP network by
  2027-03-31 and handed to the electoral commission for voting-assembly mapping and the
  electoral rolls. Source:
  [[2026-06-15-arranca-prova-de-vida-para-actualizacao-do-registo-eleitoral]].

Operational detail on the 2027 update sits on **[[angola--dpi-registry]]** and
[[registo-eleitoral-oficioso-angola]].
