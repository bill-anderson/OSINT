# SYDONIA (ASYCUDA) customs system

**The customs management system in use by Togo's revenue authority, [[otr-togo]]** — the
system of record for customs declarations and the destination for automated exemption
transmissions from **[[gest-exo]]**. **SYDONIA** is the French rendering; **ASYCUDA** is the
English form of the same system. See [[TGO]], [[dpi.govtech]].

**Scope note — deliberately thin.** The evidence held here establishes SYDONIA only as "the
customs software" that Gest-exo automatically transmits to. No version, deployment date,
operator, vendor or coverage is asserted from it. This page exists as the standing object those
facts will attach to as further sources arrive; a dated fact later pulled about or from SYDONIA
becomes a normal source citing this entity.

## Activity in this wiki

- **Automatic integration with Gest-exo (2025-09-30).** The OTR's Gest-exo exemptions platform
  provides "automatic integration with the customs software SYDONIA," with requests moving "from
  submission to automatic transmission to the SYDONIA customs system" (Atta-Kakra Essien, OTR
  Commissioner for Customs and Indirect Taxes). SYDONIA is thus the downstream system in Togo's
  clearest documented government system-to-system data exchange. See [[dpi.exchange]],
  [[gest-exo]]. Source:
  [[2025-10-01 Togo Launches Online Platform for Tax Exemptions and Waivers]].
