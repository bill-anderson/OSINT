# PeeringDB

A freely available, **user-maintained database of networks** and the go-to source
for interconnection data — the first stop in making interconnection decisions. It
catalogues **networks, Internet Exchange Points (IXPs), data centres/facilities and
carriers**, facilitating the global interconnection of networks. Non-profit,
community-driven, volunteer-run.

## Why it's a standing OSINT asset

A **[[infra.connect]]** / **[[infra.store]]** reference for mapping the physical
internet: who peers where, which IXPs and data-centre facilities exist in a given
market, and how networks interconnect. For this wiki it is useful for grounding
claims about African connectivity and data-centre ecosystems (IXP presence, carrier
density, facility footprints) against a maintained primary registry rather than
press releases.

## How to use / cite

Consult it for specific, dated facts, then file **those** as normal sources that
cite this resource — e.g. *"PeeringDB lists N IXPs / facilities in <country> as of
<date>"* becomes a `type: source` record tagging the relevant place(s) and
[[peeringdb]]. The profile itself carries no single event or date.
