# Namibia national digital ID / e-ID

Namibia's **national digital-identity programme**: a smart, contactless-microchip **electronic ID card**
holding biometric and demographic data, which the **[[ministry-home-affairs-namibia|Ministry of Home
Affairs]]** plans to launch by **July 2026** to curb identity fraud, eliminate duplicate records and enable
real-time verification. It is built on the Civil Register created by the
**[[namibia-civil-registration-identification-act-2024|CRIA 2024]]**.

The rollout is supported by **[[undp|UNDP]]** — Namibia is cited as the first user of UNDP's Model Governance
Framework and features in UNDP's "Namibia's Digital Identity Journey," which links the e-ID to the **NamX**
data-exchange layer and **Nampay** payments. Foundations trace to earlier work: a 2020 case study documented
Namibia's National Population Registration System (NPRS), Unique Identification Number (UIN), e-birth/e-death
notification and a Data Exchange Service Bus — one of sub-Saharan Africa's more advanced CRVS bases.
