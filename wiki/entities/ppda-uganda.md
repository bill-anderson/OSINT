# Public Procurement and Disposal of Public Assets Authority (PPDA), Uganda

Uganda's **public-procurement regulator**, custodian of the **Government Procurement Portal (GPP)** — noted
regionally for **publishing machine-readable procurement data**.

- **Open-contracting upgrade (Nov 2025).** A mapping report (with **CoST Uganda** and the finance ministry)
  set out upgrading the GPP to the **Open Contracting for Infrastructure Data Standard (OC4IDS)** — adding
  project-level, sustainability and environmental-safeguard data so citizens can track delivery. See
  [[uganda--dpi-govtech]], [[data.open]]. Source: [[2025-11-14-govt-upgrade-procurement-portal-oc4ids]].
