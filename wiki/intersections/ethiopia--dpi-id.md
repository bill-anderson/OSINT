# Ethiopia × Digital Identity

Ethiopia's **Fayda / National ID Program (NIDP)** is the **flagship deployment of
[[mosip]]** — the reference implementation the wider MOSIP ecosystem points to.

## State of play (mid-2026)

- **Scale:** ~**46m residents registered**, **13m cards printed**, up to **4m
  authentications/day**; **150+ agencies** performing KYC against Fayda; three active
  data-centre sites; **50+ in-house engineers**. The program charges
  **authentication-as-a-service** fees — a domestic-revenue, sustainability model other
  MOSIP adopters watch. Source:
  [[2026-07-07 Beyond the Platform Understanding the Growing MOSIP Ecosystem]].
- **Registration capacity:** expanded via the **[[safaricom-ethiopia-fayda-jv]]** (Fayda
  Lots 3 & 4) — a telco-led enrolment drive. See [[safaricom]].
- **Stack:** MOSIP is modular open-source (MPL 2.0), incubated at IIIT-Bangalore — the
  vendor-independence / [[gov.standards]] thread, read against proprietary national-ID
  stacks elsewhere.
- **Self-assessment as monitoring:** the NIDP adopted DIAL's **[[digital-id-assessment]]**
  (first run early 2025, re-run 2026) as its **official monitoring** instrument, tracking
  Fayda's improvement journey against a target of **90m registrations by 2028**. An African
  state governing its own ID programme by an open assessment framework — the
  [[capacity.research]] / [[gov.standards]] thread. Sources:
  [[2026-05-12 Strengthening Fayda Ethiopia's improvement journey through the Digital ID Assessment]],
  [[2024-01-16 Digital ID]].

## Background: the pre-Fayda ID gap

- **No national ID before Fayda (2024, DHIS2 study).** A national DHIS2 maturity assessment
  ([[dhis2-maturity-ethiopia]]) recorded that, as of its **2022 fieldwork**, Ethiopia had
  **no national digital identification program and no client registry** — the interoperability
  gap the Fayda scale-up (above) is meant to close, and a dated marker of how recent
  foundational ID is in Ethiopia. See [[dpi.mis]], [[dpi.exchange]]. Source:
  [[2024-07-26 Maturity Assessment of District Health Information System Version 2 Implementation in Ethiopia Current Status and Improvement Pathways]].

## Reads

- `sovereignty`: an African state running a nationally-governed, open-source ID at
  continental-leading scale, monetising authentication domestically — the strongest
  positive data point in the wiki's DPI-sovereignty argument.
