# Domestic finance sweep — Kenya 2024 → FY2024/25 — run 2026-07-23

**Resolution:** Kenya 2024 → **FY2024/25, 2024-07-01 to 2025-06-30** (Jul–Jun year; Kenya's first
in-scope year per the README edge note). Date window (blocks 4–7): 2024-01-01 → 2026-06-30.
Cap 40; used 14 staged (9 budget documents + 5 prose) + 3 acquisitions.

**The year's defining event shapes the whole document chain:** the Finance Bill 2024 was rejected
(Art. 115) after the June 2024 protests, stripping ~KSh 344.3bn of expected revenue. The budget was
revised **three times**: gross 4.49tn (original, Jun 2024) → 4.37tn (Supp I, Aug 2024, enacted as
Supplementary Appropriation (No. 2) Act, assented 2024-08-05) → 4.64tn (Supp II, Feb 2025) → 4.37tn
(Supp III, Jun 2025). Any FY2024/25 "allocation" without a stage label is ambiguous across four
values — stage discipline matters more here than in any year yet swept.
