# AWS adds a CloudFront edge location in Cairo

**[[amazon-web-services|AWS]]** announced a new **Amazon CloudFront edge location in Cairo**, expected to cut latency by up to ~30% on average for content delivered in Egypt. CloudFront (a distributed CDN for static/dynamic content, APIs and video) protects all edge locations against infrastructure-level DDoS with AWS Shield (WAF optional); traffic is priced within Middle East region pricing. A marker of hyperscaler edge presence building on Egypt's cable-landing position.
