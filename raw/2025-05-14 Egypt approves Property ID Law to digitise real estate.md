# Egypt approves Property ID Law to digitise the real-estate sector

Egypt's House of Representatives approved (**27 April 2025**) the **Property ID Law** (first introduced Aug 2023) — see [[egypt-property-id-law]] — mandating a **unique digital ID for every property** and a centralised database linking real-estate assets to a government platform. Each property receives a digital ID recording **location, use, ownership, licensing, violations and transaction history**, tied to Egypt's **unified digital base map** and a secure spatial information system.

A Prime Ministerial decree will name the overseeing authority (CAPMAS, MCIT and others define the structure). Physical ID plates (from the Secure and Smart Documents Complex) mark properties. **Property IDs become a prerequisite** for real-estate-registry/notary registration and for utility connections (water, electricity, gas). Tampering carries EGP 5,000–30,000 fines or up to six months' prison; national-security properties are exempt. Officials expect transparency to attract foreign and diaspora investment and enable remote transactions.
