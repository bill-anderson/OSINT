# RailTel (India) to build a data centre in Ethiopia — MEA work order

**RailTel Corporation of India** (a Government of India enterprise) received a **Ministry of External Affairs (MEA) work order** (09.01.2026) for a **data-centre project in Ethiopia** — a strategic digital-infrastructure project supporting Ethiopia's digital transformation, and a marker of **Indian** (alongside Chinese) engagement in Ethiopian data-centre build-out.
