## Description

CHEXIM loan for Phase 2 of Angola's integrated public security and anti-crime surveillance system. Extends the first phase (Integrated Centre for Public Security, CISP, opened Dec 2019) to additional cities beyond Luanda, installing CCTV, facial recognition, and command-and-control infrastructure. Part of Angola's national Safe City program using Huawei technology.
